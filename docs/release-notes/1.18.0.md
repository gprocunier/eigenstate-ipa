# Release notes 1.18.0

This release adds generic IdM inventory, vault, access-path, and sudo policy
diagnostics.

```text
New inventory behavior
  - normalized IdM host attributes
  - raw attribute companions
  - type metadata
  - structured schema warnings

New modules
  - eigenstate.ipa.vault_health
  - eigenstate.ipa.vault_artifact
  - eigenstate.ipa.access_path

New filters
  - eigenstate.ipa.ensure_list
  - eigenstate.ipa.normalize_attribute
  - eigenstate.ipa.attribute_type
  - eigenstate.ipa.sudo_risk
  - eigenstate.ipa.classify_sudo_rule
```

Release readiness requires local unit and sanity validation plus live lab
evidence for KRA/vault paths before publication.
