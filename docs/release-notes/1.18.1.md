# Release notes 1.18.1

This release carries the documentation audit fixes after 1.18.0. No collection
API surface changed.

```text
Documentation corrections

  - stronger checks for weak expected-result placeholders
  - broader Markdown example validation

Site rendering corrections
  - fluid desktop layout for reference pages
  - responsive GitHub Pages gutters
  - mobile-safe inline code and highlighted code blocks
```

Use 1.18.1 when you want the 1.18.0 collection features with the corrected
public documentation and GitHub Pages layout.
